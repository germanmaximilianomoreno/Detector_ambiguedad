# Detector de requerimientos ambiguos con Aprendizaje Profundo

Este proyecto fue desarrollado siguiendo la metodología CRISP-DM.
- En la carpeta dataset/ se encuentran los datos de entrenamiento y prueba correspondientes a los dos modelos desarrollados.
- En la carpeta src/ se incluye tanto el código de entrenamiento de los modelos como el código de la aplicación web desarrollada durante la fase de despliegue.

Podes observar la documentación completa en: https://deepwiki.com/germanmaximilianomoreno/Detector_ambiguedad
